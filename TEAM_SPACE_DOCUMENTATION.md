# Командний простір - Документація

## Огляд

Командний простір - це комплексна система для управління командою арбітражників, яка включає в себе всі необхідні інструменти для ефективної роботи та співпраці.

## Основні функції

### 1. Огляд команди (`/workspaces/[id]/team`)
- **Статистика команди**: Загальний дохід, ROI, кількість аккаунтів та кампаній
- **Учасники команди**: Список всіх учасників з їх статусом та продуктивністю
- **Остання активність**: Лента подій та дій команди
- **Майбутні зустрічі**: Планування та відстеження зустрічей

### 2. Командний чат (`/workspaces/[id]/chat`)
- **Реальний чат**: Можливість спілкування між учасниками команди
- **Статуси повідомлень**: Відстеження доставки та прочитання повідомлень
- **Групування за датами**: Зручна навігація по історії чату
- **Файли та емодзі**: Підтримка вкладень та емодзі

### 3. Управління задачами (`/workspaces/[id]/tasks`)
- **Канбан-дошка**: Візуальне управління задачами за статусами
- **Пріоритети**: Система пріоритетів (низький, середній, високий, критичний)
- **Призначення**: Розподіл задач між учасниками команди
- **Дедлайни**: Відстеження термінів виконання
- **Теги**: Категоризація задач

### 4. Календар команди (`/workspaces/[id]/calendar`)
- **Планування подій**: Створення зустрічей, дедлайнів, нагадувань
- **Типи подій**: Різні типи подій з відповідними кольорами
- **Учасники**: Призначення учасників на події
- **Онлайн зустрічі**: Підтримка посилань на відеозустрічі
- **Локації**: Вказівка місць проведення

## Архітектура

### Компоненти

#### 1. TeamChat.tsx
```typescript
interface Message {
  id: string
  senderId: string
  senderName: string
  content: string
  timestamp: Date
  type: 'message' | 'announcement' | 'system'
  status: 'sent' | 'delivered' | 'read'
  isEdited?: boolean
}
```

**Функції:**
- Відображення повідомлень з групуванням за датами
- Статуси доставки повідомлень
- Автоматичне прокручування до нових повідомлень
- Підтримка Enter для відправки

#### 2. TeamTasks.tsx
```typescript
interface Task {
  id: string
  title: string
  description: string
  assigneeId: string
  assigneeName: string
  status: 'todo' | 'in-progress' | 'done' | 'overdue'
  priority: 'low' | 'medium' | 'high' | 'critical'
  deadline: Date
  createdAt: Date
  tags: string[]
  estimatedHours: number
  actualHours?: number
}
```

**Функції:**
- Канбан-дошка з колонками статусів
- Фільтрація за статусом, пріоритетом, виконавцем
- Пошук по назві та опису задач
- Модальне вікно створення задач

#### 3. TeamCalendar.tsx
```typescript
interface Event {
  id: string
  title: string
  description?: string
  startTime: Date
  endTime: Date
  type: 'meeting' | 'deadline' | 'reminder' | 'task'
  participants: string[]
  location?: string
  isOnline?: boolean
  meetingUrl?: string
  color: string
}
```

**Функції:**
- Календарний вигляд з навігацією по місяцях
- Відображення подій на календарі
- Детальний перегляд подій за датою
- Створення нових подій

### Сторінки

#### 1. `/workspaces/[id]/team` - Головна сторінка команди
- Табова навігація між розділами
- Огляд статистики команди
- Швидкий доступ до всіх функцій

#### 2. `/workspaces/[id]/chat` - Командний чат
- Повноцінний чат з історією повідомлень
- Мокові дані для демонстрації

#### 3. `/workspaces/[id]/tasks` - Управління задачами
- Повна система управління задачами
- Мокові дані з прикладами задач

#### 4. `/workspaces/[id]/calendar` - Календар команди
- Повноцінний календар з подіями
- Мокові дані з прикладами подій

## Стилізація

### Кольорова схема:
- **Синій**: Основні елементи, активні стани
- **Зелений**: Успішні дії, завершені задачі
- **Жовтий**: Попередження, нагадування
- **Червоний**: Помилки, критичні задачі, протерміновані дедлайни

### Статуси:
- **Online**: Зелений індикатор
- **Away**: Жовтий індикатор
- **Offline**: Сірий індикатор

### Пріоритети задач:
- **Low**: Зелений
- **Medium**: Жовтий
- **High**: Помаранчевий
- **Critical**: Червоний

## Використання

### Для команди:
1. **Перегляд статистики**: Огляд продуктивності команди
2. **Спілкування**: Використання чату для координації
3. **Управління задачами**: Створення та відстеження задач
4. **Планування**: Використання календаря для планування

### Для менеджера:
1. **Моніторинг**: Відстеження прогресу команди
2. **Планування**: Розподіл задач та ресурсів
3. **Аналітика**: Аналіз продуктивності команди
4. **Координація**: Організація зустрічей та подій

Командний простір надає всі необхідні інструменти для ефективної роботи команди арбітражників, забезпечуючи прозорість, координацію та продуктивність.
